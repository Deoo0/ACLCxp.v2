## вңЁ What is ACLCxp?

> Imagine Hogwarts вҖ” but make it ACLC Tacloban.

**ACLCxp** (ACLC Experience) is a full-stack digital event management platform built for **3,000+ students** at ACLC College of Tacloban. It replaces the old-school clipboard-and-paper attendance system with QR codes, live leaderboards, AI-powered recommendations, and a full house competition system.

Built as a capstone project вҖ” but built like a real product.

## рҹҡҖ Features

### рҹҺ« QR-Based Attendance
HMAC-SHA256 signed QR codes вҖ” unique to every student, impossible to fake, expire every 24 hours. Facilitators scan, system validates, attendance recorded. Done.

### рҹ—“пёҸ Event Management
Create events with capacity limits, visibility controls (public, house-only, program-specific), waitlists, and categories. Students register, get reminders, show up, earn points.

### рҹӨ– AI Recommendations (Google Gemini)
Personalized event suggestions based on a student's interests and attendance history. Plus a chatbot that answers "when is the next CS event?" so admins don't have to.

### рҹ“Ҡ Admin Analytics
Live dashboards, exportable reports, audit logs, user management вҖ” everything an admin needs to run 100+ events a semester without losing their mind.

### рҹ”” Notifications
Email confirmations via Resend, in-app notifications, and event reminders вҖ” because students forget, and that's okay.
