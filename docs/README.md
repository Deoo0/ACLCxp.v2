# ACLCxp Documentation

This folder contains project documentation for ACLCxp.
Use the files below to understand the repository, backend API, development setup, and testing procedures.

- `PROJECT_STRUCTURE.md` — detailed repository layout and frontend/backend structure
- `API_DOCUMENTATION.md` — Django REST API endpoints and request/response examples
- `DATABASE_SCHEMA.md` — database design and model documentation
- `DEVELOPMENT_SETUP.md` — how to install and run the backend and frontend
- `TESTING_GUIDE.md` — testing workflow and mobile network setup

## Frontend Notes
The frontend is built with React, TypeScript, Vite, Tailwind CSS, and React Router.
It includes a landing page, a connectivity test page, reusable UI components, and Axios-based API integration.
